About JsonUtilities

A high-performance C# library for scanning large JSON files with byte-position tracking, JSON path extraction, and prefix-tree indexing.

Byte-Range Scanning

Extract objects from JSON collections with precise byte positions for efficient partial file reading.

JSON Path Extraction

Navigate nested structures using dot notation paths to extract specific data elements.

Trie Indexing

Build searchable indexes with prefix matching for fast lookups in large datasets.
