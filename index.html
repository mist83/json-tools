About JsonUtilities

A high-performance .NET 8 library for scanning large JSON files without full deserialization. Uses byte-range tracking, dot-notation path extraction, prefix-tree indexing, and semantic keyword search.

Byte-Range Scanning

Extract objects with precise byte positions using ArrayPool buffers and zero-copy Span<T> slicing. Parallel processing via Channel<T>.

Path Extraction

Navigate deeply nested JSON using dot-notation paths like company.departments.engineering.employees. Case-insensitive by default.

Trie Indexing

Generic prefix-tree with O(prefix_length) search. Supports unicode, duplicates, exact-match, and batch construction via TrieBuilder<T>.

Semantic Search

Index specific fields (title, cast, description) by keyword → byte offset. Search a 500 MB catalog in milliseconds. N-gram phrase search supported.

UTF-8 Validation

Validates delimiter safety for multi-byte UTF-8 sequences. Ensures {, }, ", \ bytes are never part of a multi-byte character.

Fluent API

Chainable builder: JsonTools.Scan(stream).ForCollections("products").WithHashes().WithContent().RunAsync()

Quick Start

// Byte-range scan
var result = await JsonTools.Scan(stream)
    .ForCollections("products", "reviews")
    .WithHashes()
    .WithContent()
    .RunAsync();

// Path extraction
var employees = await JsonTools.ExtractPath(stream, "company.departments.engineering.employees")
    .WithContent()
    .RunAsync();

// Semantic search index
var index = await JsonTools.BuildSemanticIndex(filePath, new SemanticIndexOptions {
    IndexedFields = ["title", "cast", "description"],
    IndexNGrams = true
});
var matches = index.SearchObjects("hanks", filePath);
