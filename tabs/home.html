Home

Load a JSON dataset to use across all tools, then choose where the work happens: hosted API, local API, or a browser-only privacy preview.

Three ways to load data

Pick a method from the sidebar:

  • Paste JSON — drop in any JSON snippet directly.
  • Upload File — load a local .json file (max 5 MB).
  • Generate Dataset — synthesize realistic fake data to stress-test the tools.

Once a dataset is active you can switch to to scan, index, and search it.

Paste JSON
